# Vagrant

https://www.vagrantup.com/

Hypervirtualiztion **must** be enabled in your BIOS!

## Installation

### Ubuntu

#### Vagrant & VirtualBox

```bash
sudo apt install virtualbox vagrant
```

```bash
vagrant --version
```
 
#### Concourse 

```bash
vagrant init concourse/lite
```

```bash
vagrant up
```
